# Splat Viewer (vorläufig)

Vorläufige Minimal-Version: Das Projekt rendert nur noch die Splat-Szene im Browser.

## Setup

```bash
npm install
npm run dev
```

## Build-Check

```bash
npm run build
```

## Splat-Datei

Standardpfad:

- `public/scene.splat`

Wenn du eine andere Datei nutzen willst, ersetze `public/scene.splat`.
