01 / LOADING

Typewriter

Ein Satz wird Buchstabe für Buchstabe getippt, mit Jitter. Unten eine Hairline mit echtem LoadingManager-Progress. Caret blinkt, Linie kollabiert zur Mitte, Cream fadet aus.

VORSCHAU IN VOLLBILD
Klick links → Loader übernimmt das ganze Fenster.
CODE — paas-loader.js / Aufruf 
// 1. Hänge den LoadingManager an alle Three.js-Loader
const manager = new THREE.LoadingManager();
const gltf    = new GLTFLoader(manager);
const tex     = new THREE.TextureLoader(manager);

// 2. Loader instanziieren — er hört automatisch auf onProgress/onLoad
const loader = new PaasLoader({
  text: "Ein Ort zum Atmen. Ein Ort für Skulpturen.",
  manager,
});
await loader.start();   // resolves wenn Reveal fertig
startScene();           // → eigene Three.js-Init
02 / CURSOR

Magnetisches Crosshair

Crosshair als Default. Wächst zu einem Ring mit „+ ÖFFNEN“ wenn ein Schild näher als 120 px ist, und gleitet hin (smoothstep-Pull). Squash entlang Velocity bei schneller Bewegung, Bouncy-Recover.

CODE — paas-cursor.js / Aufruf 
// Marker im DOM mit data-marker markieren
// (für Three.js: in jedem Frame worldToScreen → DOM-Element positionieren)
<div data-marker data-id="skulptur-01"></div>

const cursor = new PaasCursor({
  magneticTargets: () => document.querySelectorAll('[data-marker]'),
  magneticRadius: 120,
});
cursor.mount();
03 / PANEL

Schild-Detail

Klick auf den Marker → Panel kommt von unten als Glass über die Szene. Beim Scrollen (innerhalb des Panels) wird der Cream opak, die Szene tritt zurück, Phase 3 ist reines Editorial.
